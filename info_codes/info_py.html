Python 의 역사

파이썬은 1908년대 프로그래머 반 로섬이 크리스마스가 있던 주, 연구실이 닫혀 심심했던 차에 연구실이 닫혀 할 것이 없어 심심했던 차에 만들기 시작했던 프로그래밍 언어이고 파이썬이라는 프로그래밍 언어의 이름은 몬티 파이썬이라는 프로그램에서 따온 이름이라고 합니다.

파이썬 2.0버전이 2000년에 등장하며 2008년에 3.0 버전이 등장하며 여러 차례의 업데이트에 걸쳐 가장 인기 있는 프로그래밍 언어로 등극하게 되었습니다. 현재 우리가 사용하고 있는 파이썬은 Python 3.11.0입니다.
Python 의 특징

  1. "아름다운 게 추한 것보다 낫다." (Beautiful is better than ugly)
  2. "명시적인 것이 암시적인 것 보다 낫다." (Explicit is better than implicit)
  3. "단순함이 복잡함보다 낫다." (Simple is better than complex)
  4. "복잡함이 난해한 것보다 낫다." (Complex is better than complicated)
  5. "가독성은 중요하다." (Readability counts)
라는 핵심 철학에 기반하여 만든 프로그래밍 언어입니다.
Python 의 장점

1. 비전공자도 배우기 쉽습니다. C언어는 문법이 복잡하여 전공자도 다루기 어려워하는 반면 파이썬은 단순한 문법을 가지기 때문에 비전공자도 배우기 쉽습니다.
2. 다양한 분야에서 활용할 수 있습니다. 기본적인 데스크톱 애플리케이션에서 멈추지 않고 웹서버, 해킹도구, IoT, 인공지능 등 널리 활용될 수 있습니다.
3. 대부분의 운영 체제에서 동일하게 사용할 수 있습니다. C언어 등의 프로그래밍 언어는 운영체제가 제한적이며 여러 운영체제에서 사용될 수 있어도 그 사용 방법에 있어 차이점이 존재합니다. 그러나 파이썬은 모든 운영체제에서 같은 방법으로 사용할 수 있습니다.
Python 의 단점

느립니다. 쉽게 사용할 수 있는 만큼 그 속도가 다른 프로그래밍 언어에 비해 10~350배 느립니다.

이러한 단점은 현재 컴퓨터 부품의 발전으로 연산을 많이 필요로 하는 애플리케이션에서 사용하는 것이 아니라면 큰 속도 차이를 느끼지 못하기 때문에 사용 난이도가 쉬운 파이썬, 자바스크립트, 루비가 큰 인기를 끌고 있습니다.
